### üîê Steps to Construct & Send Basic Auth Headers for Jira API

1. **Generate an API token**

   * Go to [Jira API Tokens](https://id.atlassian.com/manage/api-tokens)
   * Create a new token and copy it.

2. **Form the credentials string**

   * Format: `useremail:api_token`
   * Example: `johndoe@example.com:ABC123xyz456`

3. **BASE64 encode the string**

   * **Linux / Mac / Unix**: `echo -n "user@example.com:api_token_string" | base64`
   * **Windows PowerShell**:

     1. `$Text = "user@example.com:api_token_string"`
     2. `$Bytes = [System.Text.Encoding]::UTF8.GetBytes($Text)`
     3. `$EncodedText = [Convert]::ToBase64String($Bytes)`
     4. `$EncodedText`
   * Example output: `am9obmRvZUBleGFtcGxlLmNvbTpBQlMxMjN4eXo0NTY=`

4. **Create the Authorization header**

   * Format: `Authorization: Basic <BASE64_ENCODED_STRING>`
   * Example: `Authorization: Basic am9obmRvZUBleGFtcGxlLmNvbTpBQlMxMjN4eXo0NTY=`

5. **Send the request**

   * Example curl command:
     `curl -X GET -H "Authorization: Basic am9obmRvZUBleGFtcGxlLmNvbTpBQlMxMjN4eXo0NTY=" -H "Content-Type: application/json" "https://your-domain.atlassian.net/rest/api/2/issue/QA-31"`
